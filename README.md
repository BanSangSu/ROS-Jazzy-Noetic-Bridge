# ROS Jazzy to ROS1 Noetic Bridge (Pre-built)

A pre-built [ros1_bridge](https://github.com/ros2/ros1_bridge) package for seamless communication between ROS 2 Jazzy (Ubuntu 24.04) and ROS 1 Noetic. This package eliminates the need to compile the bridge from source, allowing you to start bridging topics and services immediately.

## Features

- ✅ Pre-compiled and ready to use
- ✅ Supports standard ROS message types
- ✅ Works with ROS 2 Jazzy on Ubuntu 24.04 (Noble)
- ✅ Compatible with ROS 1 Noetic (via Docker or native installation)
- ✅ AMD64 architecture support
- ⚠️ Custom message support not included in this build

## Prerequisites
> On Ubuntu 24.04 (ROS 2 Jazzy System)

### ⚠️ Note
> This build is based on [TommyChangUMD/ros-jazzy-ros1-bridge-builder](https://github.com/TommyChangUMD/ros-jazzy-ros1-bridge-builder).
> 
> ⚠️ **For Custom Messages:** If you need to bridge custom ROS messages, please refer to the [Custom Messages documentation](https://github.com/TommyChangUMD/ros-jazzy-ros1-bridge-builder) and build your own bridge following their instructions.
